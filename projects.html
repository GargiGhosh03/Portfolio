Evaluation of Spectrum-Aided Vision Enhancer for Classification of Gastrointestinal Disease in Capsule Endoscopy

SAVE has been used to improve mucosal visualization, improving the identification of small mucosal abnormalities that conventional imaging techniques can overlook, using hyperspectral imaging techniques. The machine learning algorithms used were ResNet50, ResNet101, EfficientNetB2, EfficientNetB5 and EfficientNetV2, where EfficientNetV2 outperformed other models by producing an accuracy of 89 percent in SAVE images.

Lung and Colon Cancer Detection

Developed a deep learning model for the early detection of lung and colon cancer using medical imaging techniques. The model achieved high accuracy in identifying cancerous lesions, aiding radiologists in diagnosis. Tensorflow library has been used for implementing deep learning model. The model was successfully evaluated and the accuracy was found to be 97 percent.

Apple Watch and Fitbit Data Processing and Prediction

Data Processing was done by using pandas and numpy libraries. Models were built using the sklearn library. For data visualization seaborn and matplotlib libraries was used to create heatmaps, histplots. The accuracy predicted was 88 percent.

Warranty Claims Fraud Prediction

Developed a machine learning model to predict fraudulent warranty claims, improving the accuracy of claims processing. Accuracy was calculated under different classifiers such as Decision Tree, Random Forest, and Logistic Regression and was found to be 92 percent.
